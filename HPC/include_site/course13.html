課程簡介

最佳化方法是近數十年發展起來的，其運用數學方法研究各種系統的最佳化方案，以作為決策時的參考。在機器學習中，如何由大量訓練資料中找出最好的模型以做最佳的預測，其背後亦是滿滿的最佳化方法。了解最佳化方法，學習機器學習演算法才能知其然，亦知其所以然。

本課程重視嚴謹的數學論證，目標是以理工科系大學生具備之數學工具，從著名的支撐向量機出發，完整證明其背後的凸函數最佳化數學理論。課程內容包括

  1. 支撐向量機 (Support Vector Machine, SVM)

    2. 我們將介紹大名鼎鼎的支撐向量機，包括何謂硬/軟支撐向量機(hard/soft SVM)、如何訓練、以及何謂高斯核支撐向量機(Gaussian radial basis function kernel SVM)，及其背後的凸函數最佳化、核方法等數學原理。

  2. 拉格朗日對偶問題 (Lagrangian Dual Problem)

    3. 一個最佳化問題可以從原始問題(primal problem)與對偶問題(dual problem)去看待，為一體兩面。我們一方面以正式的數學定義何謂對偶角度，一方面提供幾何上的解釋。

  3. 弱對偶定理與強對偶定理(Weak/Strong Duality Theorem)

    4. 在甚麼樣的情形下原始問題與對偶問題其最佳化的結果是相同的？我們將以嚴謹的數學推論說清楚。透過證明強對偶定理，我們亦將解釋SVM的KKT定理。
